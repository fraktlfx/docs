---
title: "Usergroups"
---
<Info>This feature requires a subscription to Slack Pro or higher</Info>

If enabled, Usergroups will be checked to determine if a user is part of a group that is allowed to mark a post as either: Approved, Needs Revised, or CBB.

If the user is part of the usergroup associated with approvals then it will go through the standard approvals process.  
  
If a user is NOT part of the approvals usergroup, then it will create this pop-up on Slack
