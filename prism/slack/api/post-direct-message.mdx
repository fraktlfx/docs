---
title: "Post Direct Message"
description: "Posts a message to the provided user."
---

### Arguments
<ParamField path="access_token" type="string" required>
    The Bot OAuth Token acquired from the Slack app
</ParamField>
<ParamField path="slack_recipient" type="string" required>
    The ID of the Slack user
</ParamField>
<ParamField path="message" type="string" required>
    Your message that you want to send/push to the channel
</ParamField>


### Example
```python Slack\Scripts\client\slack\api\post.py lines icon="python"
slack = pcore.getPlugin('Slack')

slack.get_access_token()
project = get_current_project()
channel_id = slack.get_channel_id(access_token, project)
channel_users = slack.get_channel_users(access_token, channel_id)

prism_user = slack.get_prism_slack_username()
slack_recipient = slack.get_slack_user_id(prism_user, channel_users)

message = "Hey Slack User!"

slack.post_channel_ephemeral_message(access_token, slack_recipient, message)

```
